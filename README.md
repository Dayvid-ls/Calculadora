# Calculadora

Uma calculadora web simples e funcional, desenvolvida com HTML, CSS e JavaScript. Ela realiza operações básicas, trata números decimais (vírgula), e tem um limite de 5 casas decimais para evitar resultados muito longos.

## Tecnologias usadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Adição, subtração, multiplicação e divisão
- Suporte a números decimais (usando vírgula)
- Botão "C" para limpar tudo
- Botão "±" para alternar o sinal (positivo/negativo)
- Botão "%" para porcentagem
- Exibição com limite de 5 casas decimais
- Layout responsivo com grid

## Como executar
1. Baixe todos os arquivos (index.html, style.css, script.js)
2. Coloque-os na mesma pasta
3. Abra o arquivo `index.html` em qualquer navegador moderno

## Funcionalidades em detalhe
| Botão | Ação |
|-------|------|
| C | Limpa toda a operação e reseta o visor para 0 |
| ± | Inverte o sinal do número atual |
| % | Calcula a porcentagem (funciona com operações pendentes) |
| ÷, x, -, + | Operadores matemáticos |
| = | Exibe o resultado da operação |
| 0-9, | Digitação dos números e vírgula decimal |
